PDF repair toolkit.

Each public function accepts an input_pdf_path (and usually an output_pdf_path) and returns a dict describing what happened. All heavy lifting is deferred to optional external tools so the module stays importable even when those tools are not installed.

PDFRepairError Objects​

class PDFRepairError(RuntimeError)

Raised when a repair operation fails.

ghostscript_reprint​

def ghostscript_reprint(input_pdf_path: str,
                        output_pdf_path: str,
                        *,
                        preserve_fields: bool = False,
                        pdf_optimization: str = "prepress") -> Dict[str, Any]

Re-distill the PDF through Ghostscript.

Arguments​

  • input_pdf_path - Source PDF path.
  • output_pdf_path - Destination path for the repaired PDF.
  • preserve_fields - Whether to restore AcroForm fields after re-distilling.
  • pdf_optimization - Ghostscript PDFSETTINGS profile to apply.

Returns​

Dict[str, Any]: Metadata describing the repair result and any warnings.
